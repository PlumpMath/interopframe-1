Resumo

We present EM, an energy management tool for WSNs, which main idea is to extend the node lifetime, by adapting its duty cycle and the amount of tasks executed.

Documentation / Downloads:

  1. The use and installation manual for the EM tool on the Gadgeteer is available here.
  2. The paper EM: Energy Management Tool for Wireless Sensor Networks, presented at SBRC 2012 - Salão de Ferramentas, is available.
  3. Poster Abstract: Energy Management for Wireless Sensor Networks. Presented at 10th ACM Conference on Embedded Networked Sensor Systems (SenSys), 2012, Toronto, Ca.
  4. Manager.cs: This is the EM tool implementation for the Gadgeteer.
  5. SampleProject.rar: This is a sample project using the EM tool on the Gadgeteer.
  6. TinyOS.tar.gz: This is the EM tool implementation for the TinyOS. It includes all necessary files, as well as a README explaining how to use the tool.
